About: COVID19 Tracker World Map

Hello! Thank you for using !


This project is created by 55abhilash. It is based on the API Endpoint https://corona.lmao.ninja/countries.

This endpoint fetches latest data about COVID19 from WorldOMeter, which is a reliable source for statistical information.
